Professional Summary

For the past 3 years, I have been studying Computer Technologies Engineering. Throughout my studies, I have contributed to the development and growth of various projects, resulting in a satisfactory outcome each semester. I have participated in the creation of a temperature and humidity sensor using IoT, developed a 2D video game and a website for a percussion association called "Percussive Arts Society" using Unity 2D, created a local network using Cisco equipment and Packet Tracer software, worked with data structures and algorithms to solve various course projects, and recently, in my fifth semester, developed a mobile application for Android devices for the government of Atizapán de Zaragoza using Android Studio and Kotlin. I also had an introduction to AI, specifically using Python agents to develop a smart city with a Python module called Mesa. Additionally, I utilized Flask to establish a connection for a more realistic simulation and Unity in 3D for the entire simulation. It is important to note that documentation is always required for these projects, as they are delivered to a client (training partner) upon completion.

I possess skills in software development, teamwork, effective communication, synthesis, adaptability, and customer orientation. Additionally, I have knowledge in various programming languages such as C#, HTML, CSS, Python, Javascript, PHP, Typescript, and C++. I have experience in software documentation and the use of cloud tools like Azure, AWS, and DigitalOcean, as well as in operating system administration like Ubuntu and repository management on Github.
